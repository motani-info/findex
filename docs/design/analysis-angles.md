## 切り口① YoC（取得利回り）× 増配持続性 — 「本物の増配株 vs 一過性の罠」

**ステータス**: 検証済み（2026-06-14・旧DB実データ）。有望。
**着想（ユーザー）**: 1万円投資して当初3%(300円)だった配当が、増配で数年後に15%(1500円/取得利回り)になっている会社を見つけたい。ただし**好景気やブーストで一時的に増えただけの可能性**があるので、健全性と組み合わせて評価したい。

### A. 量の指標：YoC（Yield on Cost＝取得利回り）
- 定義: `YoC_Ny = 最新の年間配当(DPS) ÷ N年前の株価`（N=3,5,10）。「当時1万円で買った人が、今いくら受け取るか」
- 補助: `dividend_multiple_Ny = 最新DPS ÷ N年前DPS`（dividend_annualだけで算出可）
- **CAGR/倍率より優れる理由**: 倍率は低ベースを過大評価する（実証: イクヨ 0.5円→33円=66倍、日本製鉄 市況回復36倍が上位を占める）。YoCは「当時の配当が極小なら取得利回りも低い」ので**低ベースノイズを自動で抑制**する。
- データ要件: N年前の株価（price_history・**2000年以降のみ**。株価バックフィルが前提＝クラスC）。

### B. 質の指標：増配持続性スコア（DSS）— なぜ増えたかを問う
**核心の分解式（実証済み）**:
```
DPS倍率 ＝ EPS倍率 × 配当性向の変化倍率
```
増配が**EPS（利益）牽引なら持続的**、**配当性向の拡大頼みなら頭打ち/一過性**。

実証（5年・DPS2倍以上、旧DB）:

| 銘柄 | DPS倍 | EPS倍 | 判定 |
|---|---|---|---|
| 霞ヶ関キャピタル | 24x | 7.9x | EPS牽引＝**健全** |
| ジャパンエンジン | 14.7x | 8.0x | EPS牽引＝健全 |
| イクヨ | 66x | 0.0x | EPS無し＝**性向拡大の一過性** |
| 日本製鉄 | 36x | 0.5x(減) | EPS低下＝**市況回復の罠** |
| THK | 16.4x | −3.6x(赤) | 一過性 |

→ 同じ高YoCでも**EPSが伴うものだけが本物**。ブースト株を弾ける。

### C. 持続性の評価観点（一般論＝ほぼ教科書通り。ユーザー主観と一致）
| 観点 | 見る指標 | 意味 |
|---|---|---|
| 原資が利益で増えているか | EPS成長（分解式のEPS牽引度） | 性向拡大頼みでない |
| 払い過ぎていないか | 配当性向 | 高すぎ＝増配余地なし/減配リスク |
| 現金で賄えているか | FCF配当カバレッジ | 借金・資産売却でなく実キャッシュ |
| 不況に耐える体力 | ネットキャッシュ・現金 | 下落局面で配当を守れる |
| 還元方針へのコミット | **DOE**（=ROE×配当性向） | 利益が振れても自己資本は安定→持続性シグナルとして配当性向より優秀。日本で採用増。**現16指標に無く追加候補** |
| 一過性でないか | 連続増配年数（既存） | 毎年か単発か |

### D. 組み合わせ方：「YoC（量）× DSS（質）」の2軸
```
        持続性 高
          │ ②育成株        ① 本命
          │ (YoCはまだ低い   (高YoC×高持続=
          │  が健全)          狙うべき増配株)
──────────┼──────────────── YoC 高
          │ ④地味          ③ 一過性の罠
          │                 (高YoCだがEPS伴わず
        持続性 低             =イクヨ/日鉄)
```
- **①右上が本命、③右下が罠**。YoC単独だと③に騙される（＝ユーザーの懸念そのもの）
- 使い方候補: (a)2軸分類で見せる (b)「質調整YoC＝YoC×DSS」でランキング (c)持続性ゲート（DSS基準未満はYoC無効）。**方式はD4.5で確定**
- 投稿フックとして優秀: 「増配率は高いが危ない株 vs 本物の増配株」は興味を引く

### E. 留意（実装時）
- 特別配当がDPSを膨らませる → 普通配当と特別配当を分離（J-Quants DivAnnの内訳確認）
- 市況ピーク益での一時的高配当 → EPSの自己トレンド比（循環株検出）も併用余地
- N年前株価が要る → 株価2000年遡及（クラスC）が前提

→ **D4.5（指標較正）**でYoC・DSS・DOEを指標化、**D5（X発信）**で2軸の投稿テーマ化。10年CAGRはこの切り口で置換/降格を検討。
